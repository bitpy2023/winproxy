# Proxy Automation 🚀

یک برنامه پایتونی سبک و مدرن برای تست اتوماتیک پروکسی‌ها و تنظیم بهترین آنها در ویندوز.

## ✨ ویژگی‌ها

- 🎨 **رابط کاربری مدرن** - دایره GO زیبا در مرکز صفحه
- ⚡ **تست سرعت بالا** - تست همزمان با سرعت فوق‌العاده  
- 🔄 **اتوماسیون کامل** - پیدا کردن و تنظیم خودکار بهترین پروکسی
- 📊 **نمایش وضعیت زنده** - نتایج تست به صورت Real-time
- 🔔 **نوتیفیکیشن هوشمند** - اطلاع‌رسانی محو شونده
- 💾 **خروجی نتایج** - ذخیره نتایج در فایل JSON

## 🛠️ نحوه استفاده

1. فایل `proxies.txt` را با لیست پروکسی‌های خود پر کنید
2. برنامه را اجرا کنید: `python proxy_automation.py`
3. روی دایره **GO** کلیک کنید
4. برنامه به صورت خودکار بهترین پروکسی را پیدا و تنظیم می‌کند

## 📝 فرمت فایل پروکسی

#
ip:port
192.168.1.1:8080
10.0.0.1:3128
proxy.example.com:8080


## 🎮 کلیدهای کنترل

- **GO** - شروع اتوماسیون
- **Clear Proxy** - پاک کردن تنظیمات پروکسی
- **Export Results** - ذخیره نتایج
- **Exit** - خروج از برنامه

## ⚙️ نیازمندی‌ها

- Python 3.7+
- Windows 10/11
- دسترسی Administrator (برای تنظیم پروکسی سیستم)
